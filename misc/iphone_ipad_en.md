## iPad/iOS Guide
1. Your device must run iPadOS 15 / iOS 15 or later.
2. Install Userscripts from the App Store. Installation guide: https://github.com/quoid/userscripts#installation
3. In `Settings -> Safari -> Extensions`, enable Userscripts.
4. In the Files app, create a subfolder and place the eHunter script file in it. Download: https://github.com/hanFengSan/eHunter/releases
5. Open the Userscripts app and select the subfolder you just created.
6. Open e-hentai (not exhentai) in Safari and enter any gallery page.
7. On a gallery page, tap the extension icon to the left of the refresh button in the address bar, allow Userscripts for this site, then tap the new button and make sure eHunter is enabled.
8. Refresh the page to use eHunter. (nhentai may require refreshing twice.)
